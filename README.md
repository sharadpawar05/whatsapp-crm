## ✨ Features

- 💬 **Conversation Management** - View and respond to WhatsApp messages
- 👥 **Contact Management** - Organize and tag contacts
- 📊 **Analytics Dashboard** - Track response times, message volume
- 🔄 **Automated Responses** - Set up quick replies and chatbots
- 📱 **Mobile Responsive** - Works on all devices
- 🔐 **Secure Authentication** - Protected routes and API

## 🚀 Quick Start

### Prerequisites

- Node.js 18+
- WhatsApp Business API credentials

### Installation

```bash
# Clone the repository
git clone https://github.com/sharadpawar05/whatsapp-crm.git
cd whatsapp-crm

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your WhatsApp API credentials

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
